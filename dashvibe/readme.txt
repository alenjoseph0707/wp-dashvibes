Transform your WordPress admin dashboard with fun seasonal themes — Snowfall, Pizza Party, Galaxy & more!

== Description ==

DashVibe brings life to your boring WordPress admin dashboard with beautiful, animated visual themes.

**Available Themes:**

* ❄️ **Snowfall** — Falling snowflakes, icy blue palette & frosted glass effects
* 🍕 **Pizza Party** — Floating pizza slices, warm colors & cheesy vibes
* 🌌 **Galaxy** — Twinkling stars, shooting stars & cosmic nebula

**Features:**

* One-click theme activation from a beautiful dashboard
* Quick-switch from the WordPress admin bar
* Lightweight HTML5 Canvas animations (no heavy libraries)
* Custom color schemes for the entire admin
* Admin-only — your front-end website is never affected
* Clean, well-documented code following WordPress coding standards

== Installation ==

1. Upload the `dashvibe` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **DashVibe** in the admin sidebar to choose your theme
4. Or use the 🎨 DashVibe menu in the admin bar for quick switching!

== Frequently Asked Questions ==

= Does this affect my website's front-end? =

No! DashVibe only modifies the WordPress admin dashboard. Your visitors will never see any changes.

= Will this slow down my admin? =

No. All animations use lightweight HTML5 Canvas with requestAnimationFrame — it's GPU-accelerated and very performant.

= Can I have different themes per user? =

The current version uses a global setting. Per-user themes are planned for v2.0!
